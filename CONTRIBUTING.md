# Contributing Guide

Thank you for your interest in contributing to the **MotionSites.ai Prompt Library**! 🚀

## 📋 Ways to Contribute

### 1. Submit a New Prompt
Have a great AI web design prompt? We'd love to include it!

**Requirements for new prompts:**
- Must generate an **original, fictional** design (no real brand clones)
- Must be optimized for **React + Tailwind CSS**
- Must include clear specifications (colors, layout, typography, animations)
- Should produce a visually impressive result

### 2. Improve Existing Prompts
- Fix typos or unclear descriptions
- Optimize prompt structure for better AI output
- Add missing specifications

### 3. Report Issues
- Broken demos
- Incorrect prompt formatting
- Missing files

## 🔧 How to Contribute

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR-USERNAME/motionsites-prompt-collection.git
   ```
3. **Create a branch**:
   ```bash
   git checkout -b feature/my-new-prompt
   ```
4. **Make your changes** following the project structure
5. **Commit** with a clear message:
   ```bash
   git commit -m "feat: add [Design Name] prompt"
   ```
6. **Push** and open a **Pull Request**

## 📝 Prompt File Format

Create a new `.md` file in the `prompts/` directory:

```markdown
# Design Name

> **Category:** SaaS / Agency / Portfolio / etc.
> **Type:** Hero / Landing Page / Component
> **License:** free

---

## 📋 Prompt

\`\`\`text
Your detailed prompt here...
\`\`\`

---

*Generated by MotionSites Export Tool*
```

## ⚠️ Important Guidelines

- **NO copyrighted content** — Do not submit prompts that clone real brands
- **NO trademarked names** — All brand names in prompts must be fictional
- **Original work only** — Do not copy prompts from other sources
- Write clean, readable prompt descriptions
- Follow the existing file naming convention (`Design_Name.md`)
- Test your prompt with at least one AI tool before submitting

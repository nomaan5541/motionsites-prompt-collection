## 1. Statutory Authority — 17 U.S.C. § 107

Under **Section 107 of the Copyright Act of 1976**, allowance is made for **"Fair Use"** for purposes such as criticism, comment, news reporting, teaching, scholarship, education, and research.

Fair Use is a doctrine in United States copyright law that permits limited use of copyrighted material without requiring permission from the rights holders under specific conditions.

---

## 2. Four Factors Analysis of Fair Use

### 1. Purpose and Character of the Use
This repository is an **open-source, non-commercial educational library**. The prompts provided are **highly transformative**:
- They do not copy, mirror, or redistribute proprietary source code or private visual files.
- Instead, they formulate natural-language instructions describing generic UI layout structures, color theory, typography hierarchy, and animation concepts for AI coding models.

### 2. Nature of the Copyrighted Work
Web layout patterns, responsive grid structures, color gradients, and common UI component paradigms (such as hero sections, pricing cards, footers, and modal popups) represent standard industry design conventions and functional UI building blocks.

### 3. Amount and Substantiality of the Portion Used
Only high-level design pattern references and public visual inspiration are used to structure descriptive text prompts. No raw backend databases, closed-source UI libraries, or proprietary asset packages are hosted or redistributed.

### 4. Effect upon the Potential Market or Value
This educational repository does **not** harm or displace the market for any commercial product, template marketplace, or design software. Rather, it promotes frontend design literacy and teaches developers how to write structured prompts for modern AI coding tools.
